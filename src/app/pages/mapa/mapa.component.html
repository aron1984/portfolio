Map using Mapbox

En este ejemplo de Mapbox vemos un emplazamiento de geometría punto, con un popup. Además, añadí una capa de tres puntos que fueron importadas en Mapbox desde un archivo GeoJson generado (Digitalizado) en QGIS.

Nótese que el marcador principal se puede mover por el mapa (Faltaría añadir una función que nos diga las coordenadas de ese marcador).

Map using Leaflet

En cambio, en Leaflet lo que hice fue generar dos simples marcadores con codigo JavaScript, seteando las coordenas, el símbolo, y el popup; pero uno de ellos lo podemos desplazar con el mouse. Además creé esos círculos, como zona de influencia a modo de ejemplo.

También use la variante de archivo GeoJson, pero en este caso convirtiéndolo en una variable. Y desde otro componente (uso Angular), lo importé. Setié las propiedades (estilos de simbología) y se añadieron al mapa.
